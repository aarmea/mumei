  int color: the color of the robot's eyes
    colors map to numbers
      (0, 1, 2, 3) = (Gray, Red, Green, Blue)
  
  int dir: the direction of the robot
    directions map to numbers
      0 = not moving
      1 = left
      2 = right

  int x: the robot's x coordinate (read-only)
    0 = leftmost position
