int *x: an integer pointer that represents the robots's 
        x coordinate  (  0 - starting position  )

int *y: an integer pointer that is the robots's y 
       coordinate ( 0 - starting position  )

int *disty: an integer pointer that points to how 
            high the robot needs to climb

int *c: an integer pointer representing
       the robot's eye color
  0 - Gray (default) 
  1 - Red
  2 - Blue
  3 - Green
